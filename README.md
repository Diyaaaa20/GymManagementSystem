## Gym Management System

A simple Java-based Gym Management System that allows gym members to log in and check their membership status, including expiry date and renewal information.

---

## Project Description

This project is built using **Core Java (OOP concepts)**.  
It simulates a gym membership system where:

- Members can log in
- Membership expiry date is calculated
- Remaining days are displayed
- Renewal message is shown if membership is about to expire
